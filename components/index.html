Main Features

Why choose this template and what are the benefits?

Rich and diverse components

The template has built-in function pages, price pages, contact information and comment components, and supports Ant Design (antd) style, allowing you to easily create a website with full functions and beautiful interface.

Optimize SEO

The template is designed with a friendly structure, which helps improve the search engine indexing effect. At the same time, `robots.txt` file configuration is supported to disable specific pages to ensure that search rankings will not be negatively affected.

Support MDX

Blog posts are in MDX format, which allows the use of custom components, supports code highlighting, and flexibly handles various MD syntaxes. After building, the corresponding HTML file is automatically generated

Support internationalization

Write once, output in multiple languages, generate corresponding files for each language

Next.js 13 native support

Supports the latest version of Next.js and can use various native features of nextjs

Free and Open-Source

The theme of this blog is permanently free and the source code is open and shared.
